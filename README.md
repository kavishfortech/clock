# IST Digital Clock

A minimalist, high-contrast digital clock designed for **Indian Standard Time (IST)**. This project follows a "nothing else" aesthetic, featuring clean white typography on a deep black background.

## 🚀 Features
- **Accurate IST:** Uses the `Intl.DateTimeFormat` API to ensure the time is correct regardless of the local system settings.
- **Minimalist Design:** Pure white-on-black interface with zero distractions.
- **Single File:** Everything (HTML, CSS, JS) is contained within one file for easy deployment.
- **Responsive:** Fluid typography that scales for mobile and desktop screens.

## 🌐 Live Preview
Check out the live website here: [kavishfortech.github.io/clock](https://kavishfortech.github.io/clock)

## 🛠️ Tech Stack
- **HTML5**
- **CSS3**
- **Vanilla JavaScript** (No libraries required)

## 📦 Installation & Usage
1. Clone this repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.
